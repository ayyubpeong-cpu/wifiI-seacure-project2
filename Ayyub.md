# WiFi Secure Project

## Deskripsi
WiFi Secure Project adalah project pembelajaran yang bertujuan untuk memahami
dasar-dasar keamanan jaringan WiFi, seperti pengenalan enkripsi, autentikasi,
dan praktik keamanan jaringan.

## Tujuan
- Memahami konsep dasar keamanan WiFi
- Mempelajari jenis-jenis enkripsi WiFi (WEP, WPA, WPA2, WPA3)
- Simulasi penerapan keamanan jaringan secara etis

## Teknologi yang Digunakan
- Python
- Linux
- Tools jaringan (untuk pembelajaran)
